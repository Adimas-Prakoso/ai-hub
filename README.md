## :toolbox: Getting Started

### :gear: Installation


Install All Library
```bash
pip install -r requirements.txt
```



### :running: Run Locally

Clone the project

```bash
https://github.com/Adimas-Prakoso/ai-hub
```

Start APP
```bash
python main.py
```
